Discountify ...in Progress

Discountify is the first project I began working on prior to learning best practices. The site at present is used to illustrate a concept of having an easy to use one page database of referral codes which enable users to get discounts. The roadmap of this project is to ultimately implement the ability for users to implement their own set of referral codes on a static shareable url.
